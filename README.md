## Install
### Using `lazy.nvim`
```lua
{ 'devoc09/lflops.nvim' }
```

### Using `vim-plug`
```vim
Plug 'devoc09/lflops.nvim'
```

## Config
Inside `init.vim`
```vim
colorscheme lflops
```

Inside `init.lua`
```lua
vim.cmd('colorscheme lflops')
```
